# React + Typescript + Mobx Templates
Simple and Clean Web+Electron Architecture

# Current Builds Environment
```
npm 6.4.1
node v9.11.2
```

# Init
```
npm install
```

# Run Web
On your local, run
```
npm run start
```

or 

```
npm run build:dev
npm run build
```
to build static scripts

# Run Electron
```
electron dist/index.html
```
